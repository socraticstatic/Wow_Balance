# Design Context - Balance Druid Dossier

## Target Audience
Serious World of Warcraft players who main Balance Druid. They know the spec. They want fast, accurate, opinionated information - not a wiki. Think of it as the difference between a player reading Wowhead (reference manual) and reading a pro player's personal notes (curated, decisive, no hedging).

## Use Cases
- Check current BiS gear before raid night
- Verify rotation priority for a specific build
- Compare talent loadouts across content types
- See what top-ranked players are running this week
- Quick-reference stat priority, enchants, gems
- Browse transmog sets and druid forms for fashion

## Brand Personality / Tone
Editorial. Opinionated. Data-forward. Like a Bloomberg terminal met a gaming magazine. Dark, high-contrast, precise. The solar/lunar duality of the Balance spec is the design DNA - gold/amber for solar themes (gear, stats, rankings), indigo/violet for lunar themes (talents, builds, abilities), green for nature themes (forms, lore, racials). NOT applied as garish color-coding. Applied as subtle accent shifts.

## Differentiation
This is the only Balance Druid resource that:
1. Derives its data directly from live API sources (WarcraftLogs, Blizzard, Raider.io)
2. Is built for ONE spec only - maximum depth, zero breadth
3. Looks like an award-winning editorial site, not a game fansite
4. Updates weekly with fresh rankings and meta shifts
